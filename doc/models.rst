|                           | regularized                            |

| multicomponent models     | Shan-Chen                              |
|                           +----------------------------------------+
|                           | free energy [PRE78]_                   |
+---------------------------+----------------------------------------+
| turbulence                | Smagorinsky LES                        |

|                           | ELBM                                   |
+---------------------------+----------------------------------------+

|                           | shallow water                          |

|                           | checkpointing                          |

|                           +----------------------------------------+
|                           | VTK                                    |
+---------------------------+----------------------------------------+
| computational backends    | CUDA                                   |
|                           +----------------------------------------+
|                           | OpenCL                                 |
+---------------------------+----------------------------------------+

|                           | velocity and density                   |

|                           | 1D profiles of correlations of         |
|                           | velocity components and density        |
|                           +----------------------------------------+
|                           | total kientic energy and enstrophy     |
+---------------------------+----------------------------------------+
